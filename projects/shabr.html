SHABR

In this project we were supposed to create a website for a hot air balloon company.

For this specifical project we had to show our knowledge for the bootstrap tools we have learned. I had choosen buttons, images, and shadows. I applied images in the gallery part of the page to make the gallery of the pictures more organized. I used buttons to apply links of one part of the page to the other. I tried to use shadows on the name of the website but then I realized it didnt't look so well, so I removed it. My greatest hardship for this project was applying a grid to my wwebsite, I was successful in applying it to all the other pages besides the home page. Personally, I think this was one of the hardest projects for this year. I spend a lot of time in this proejct that I havent for previous projects and I still feel that my final result could be better.
